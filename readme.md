# Dev Form Filler

## Running the Project

#### Linux/macOS:

```bash
$ pnpm dev
```

or to open extension as a web page

```bash
$ pnpm start
```

#### Windows:

```bash
$ pnpm dev:win
```

or to open extension as a web page

```bash
$ pnpm start:win
```

### After loading the extension into the browser

1. Navigate to the `example` folder:

```bash
$ cd example
```

2. Run the project in development mode:

```bash
$ pnpm dev
```
